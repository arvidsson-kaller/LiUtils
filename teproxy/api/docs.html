LiUtils TimeEdit Proxy API Documentation

Proxy for accessing TimeEdit resources without scrambling.
Used in https://liutils.se
For more LiUtils API, see https://be.liutils.se

File Extension Redirect Action
.html Redirects to the TimeEdit website
.ics Redirects to an iCalendar file

Note: All endpoints return a redirect to the TimeEdit URL.

/:courses

This endpoint retrieves information about multiple courses.

Parameters:

  • :courses (string): A list of course codes separated by +.

Example:

/tddd38+tddd27
/tddd38+tddd27.ics

Note: All endpoints support .ics and .html.

/room/:rooms

This endpoint retrieves information about multiple rooms.

Parameters:

  • :rooms (string): A list of room names separated by +.

Example:

/room/su01+su02
/room/su01+su02.ics
